## Board Vs Page Background

There are two different visual backgrounds in the client:

- The board texture inside `.goban` uses `app/img/wood-board.jpg`.
- The area around the board is styled by `.goban-page` and uses CSS radial gradients, not an image file.

## Grid Color

The board grid is not defined by an image asset.

- Grid lines are drawn on `grid-canvas` in `app/js/gopanda.js`.
- The grid line color is hardcoded as `black` in the canvas rendering code.
- Coordinate labels are also canvas-drawn and use `rgba(0,0,0,0.5)`.

That means changing grid color is a code patching task, not an asset replacement task.

## Notes On Sabaki Compatibility

Sabaki themes are still useful as an import source because they package board and stone visuals plus CSS describing how they are used. Pandanet does not consume the Sabaki theme directly, so this project translates from either direct asset files or a Sabaki package into Pandanet's concrete asset files.
